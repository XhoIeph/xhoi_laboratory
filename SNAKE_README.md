# 贪吃蛇游戏 - 编译和运行指南

## 前置要求
- Windows 系统
- GCC 编译器（推荐使用 MinGW）

## 编译方法

### 方法1：使用 GCC（推荐）
打开命令行（PowerShell 或 CMD），进入文件所在目录，运行：
```bash
gcc snake_game.c -o snake_game.exe
```

### 方法2：使用 Dev-C++
1. 打开 Dev-C++
2. 文件 → 打开 → 选择 snake_game.c
3. 运行 → 编译运行（或按 F11）

### 方法3：使用 Visual Studio
1. 创建新的 C++ 控制台项目
2. 将代码复制到项目中
3. 按 Ctrl + F5 运行

## 运行游戏
编译成功后，双击 `snake_game.exe` 或在命令行输入：
```bash
.\snake_game.exe
```

## 游戏操作
- **W** - 向上移动
- **S** - 向下移动
- **A** - 向左移动
- **D** - 向右移动
- **X** - 退出游戏

## 游戏规则
1. 吃到食物（*）增加分数和长度
2. 撞墙或撞到自己会游戏结束
3. 每吃一个食物得10分

## 常见问题

### Q: 编译时出现错误？
A: 确保使用的是 Windows 系统，因为代码使用了 Windows 特定的函数。

### Q: 无法找到 conio.h？
A: 使用 MinGW-w64 或 Dev-C++，它们包含此头文件。

### Q: 如何调整游戏速度？
A: 修改代码中的 `Sleep(100)`，数字越小速度越快。

### Q: 如何调整游戏区域大小？
A: 修改代码开头的 `WIDTH` 和 `HEIGHT` 常量。

## 学习建议
1. 先运行程序，体验游戏
2. 阅读代码注释，理解每个函数的作用
3. 尝试修改参数（如速度、地图大小）
4. 逐步添加新功能（如难度提升、障碍物等）

## 代码学习重点
- **结构体**：理解如何用 Point 表示坐标
- **数组**：蛇身用数组存储
- **循环**：游戏主循环和绘制循环
- **条件判断**：碰撞检测
- **函数**：模块化编程
